# Ashok Vangapandu Portfolio

A static portfolio for Ashok Vangapandu, Senior Software Engineer at PLM Indistech. The site showcases product design, frontend systems, AI-assisted workflows, resume details, testimonials, and detailed project case studies.

## Features

- Responsive portfolio homepage with animated sections and interactive cards
- Dedicated resume page
- Project case study pages for Medi Connect, Vantage Nutrition R&D, Field Survey Personnel, Edith Design System, and Smart Dashboards
- Smooth scrolling, active navigation states, reveal animations, magnetic interactions, and mobile navigation
- Organized assets, CSS modules, JavaScript entrypoint, and scalable page folders
- Static hosting compatibility for GitHub Pages

## Tech Stack

- HTML5
- CSS3 with modular imports
- Vanilla JavaScript
- Static assets hosted from the repository
- GitHub Pages compatible deployment

## Local Setup

No build step is required.

1. Clone the repository.
2. Open `index.html` directly in a browser, or serve the folder with any static server.
3. Navigate through the homepage, resume page, and project case studies.

Optional local server:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Deployment

This project can be deployed as a static site.

For GitHub Pages:

1. Push the repository to GitHub.
2. Open repository settings.
3. Enable Pages from the main branch root.
4. Visit the generated GitHub Pages URL.
